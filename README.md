## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Aetheridon/conch-shell
   ```

2. Navigate to source directory:
   ```bash
   cd conch-shell/src
   ```

3. Build the project:
   ```bash
   make
   ```

4. (Optional) Clean build files:
   ```bash
   make clean
   ```

## Usage

Run Conch from the terminal:
```bash
./conch
```
